# ra3-pages

Este repositorio contiene la práctica realizada para aprender a crear y publicar un sitio web estático utilizando GitHub Pages.

## Descripción

El objetivo de esta práctica es comprender cómo funciona el despliegue de páginas web estáticas desde un repositorio de GitHub, así como organizar correctamente los archivos HTML, CSS y JavaScript que componen el sitio.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages
- Visual Studio Code

## Estructura del proyecto
-index.html : estructura del contenido de la página.
-styles.css :estilos visuales del sitio.
-script.js : animación de partículas en el fondo.
-README.md : archivo informativo
## Publicación del sitio

La página web se publica automáticamente mediante **GitHub Pages**.

Cada vez que se actualizan los archivos en el repositorio, GitHub vuelve a desplegar el sitio.
