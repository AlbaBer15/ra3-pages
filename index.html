Práctica RA3

Publicación de sitios estáticos y documentación online.

Objetivo

El objetivo de esta práctica es aprender a crear y publicar una página web estática utilizando GitHub Pages, organizando los archivos HTML, CSS y JavaScript.

Herramientas utilizadas

  • GitHub
  • GitHub Pages
  • HTML
  • CSS
  • JavaScript
  • Visual Studio Code

Pasos realizados

  1. Crear un repositorio en GitHub
  2. Crear el archivo index.html
  3. Diseñar la página con HTML y CSS
  4. Añadir efectos visuales con JavaScript
  5. Publicar el sitio mediante GitHub Pages

Conclusión

GitHub Pages permite publicar sitios web estáticos de forma sencilla sin necesidad de configurar servidores, siendo una herramienta muy útil para proyectos académicos y documentación.
